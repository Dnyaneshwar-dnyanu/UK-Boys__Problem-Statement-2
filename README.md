# Kannada Word Builder & Validator

A smart system that joins Kannada words following grammatical rules (Sandhi) and validates the results.

## Features

- **Dictionary Validation**: Checks if words exist in the database
- **Sandhi Application**: Applies Kannada grammar rules for word joining
- **Case Marker Addition**: Adds appropriate vibhakti endings
- **Fuzzy Matching**: Suggests corrections for typos
- **Smart Hints**: Suggests common word combinations
- **Compound Validation**: Validates if joined words make sense
